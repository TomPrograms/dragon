# Basic Data Types

Dragon data types, like most high-level languages, are dynamic (their type can change) and come in broad categories. The basic data types can be seen below.

```js
1 // number
1.1 // number
"hello" // string
'hello' // string
[1, "2" , 3] // list
// dictionary
{
  test: 'test',
  test2: 1
}
null // null
```
